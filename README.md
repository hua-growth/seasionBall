# 安装环境说明

## 目录结构

```
项目根目录/
├── index.html          # 前端页面（原生 JavaScript 版本）
├── app.js              # 前端 Vue.js 版本脚本
├── hou/                # 后端 Java Spring Boot 项目
│   ├── pom.xml         # Maven 依赖配置
│   └── src/
│       └── main/
│           ├── java/   # Java 源代码
│           └── resources/
│               └── application.yml  # 应用配置文件
└── INSTALL.md          # 本文档
```

---

## 一、系统要求

### 后端环境要求

- **Java**: JDK 11 
- **Maven**: 3.6.0 
- **MySQL**: 5.7
